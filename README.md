A lightweight RESTful API built with **FastAPI** and **SQLModel** for managing marketing campaigns.  
This service provides CRUD operations for creating, retrieving, updating, and deleting campaign records stored in a local SQLite database.

 Features
- FastAPI-based asynchronous web service
- SQLite + SQLModel ORM integration
- Auto-creation of database tables on startup
- Sample seed data injected on first run
- Generic API response wrapper
- CRUD endpoints for campaigns
